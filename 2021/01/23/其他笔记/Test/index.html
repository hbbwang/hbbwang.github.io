代码块


代码块的语法和原来的差不多，也是使用 ``` ，不过默认没有行号，需要指定，也可以指定语言，不过插件只包含了C like，css和js这些常见的，没有太多其他语法支持。
```必须顶格，前面不能有其他任何字符 例子：
```c,line-numbers
float a = 1.0;
float b = a + b;
```

拓展写文章的功能，都是映射html的style ，css语法：


颜色的字体,支持十六进制,缩写col、c


例如:
\[color(255,0,0)]红色字体： 红色字体。
\[c(255,0,0)]紫色字体： 紫色字体。
\[c(#FFDAB9)]肤色字体： 肤色字体。
需知:标题(h)不支持变色，也不打算支持。
其他：
黄色字体。
蓝色字体：[c(0,255,0)] 绿色字体。
其他颜色字体。

字体大小设置,缩写s


和上面的颜色一样，通过[]设置：
\[size(10px)]字体大小10px：字体大小10px。
\[s(50px)]字体大小50px：字体大小50px。

下划线,缩写u


接着上面的，也是支持组合的，顺序随机。
\[underline()]下划线，不带属性也行：下划线，不带属性也行。
\[u(red dashed)]当然也可以带颜色和类型的属性：当然也可以接颜色和类型的属性，这是一个红色不连续的下划线。

支持组合


\[c(255,255,0),s(25px)]黄色字体大小25px： 黄色字体大小25px。
\[c(0,255,0),s(15px),u()]下划线绿色色字体大小15px： 下划线绿色色字体大小15px。

字体背景色,支持十六进制,缩写b、bg


\[background(10,180,0)]绿色背景： 紫色背景
\[b(120,100,0)]橙色背景： 橙色背景
\[b(#FFDAB9)]肤色背景： 肤色背景
